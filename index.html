  • Nearest Neighbor

    Starts at one city and connects with the closest unvisited city. It repeats until every city has been visited. It then returns to the starting city.

  • Shortest Edge Insertion

    Inserts shortest edge possible. This is repeated until we have a cycle containing all of the cities.

  • Nearest Insertion

    Repeatedly finds the city not already in the tour that is closest to any city in the tour, and places it between whichever two cities would cause the resulting tour to be the shortest possible

  • Farthest Insertion

    Repeatedly finds the city not already in the tour that is furthest from any city in the tour, and places it between whichever two cities would cause the resulting tour to be the shortest possible.

  • Random Insertion

    Selects random city, inserts it between closest two cities which are already visited.

  • 2-Opt Algorithm

    Tries to improve a tour obtained with some other heuristic by switching 2 edges at one time.
